# 测试思考块识别

这是一个测试文档，用于验证各种思考块标签的识别和折叠功能。

## 1. 英文思考块
<thinking>
这是英文思考块的内容。
它应该被正确识别和折叠。
包含多行内容。
</thinking>

## 2. 中文思考块
<思考>
这是中文思考块的内容。
它应该被正确识别和折叠。
也包含多行内容。
</思考>

## 3. 思索标签
<思索>
这是思索标签的内容。
它应该被正确识别和折叠。
这是另一种中文思考块。
</思索>

## 4. 混合语言标签
<thinking思考>
这是混合语言标签的内容。
开始和结束标签都包含两种语言。
</thinkingthinking>

## 5. 跨语言标签

这是跨语言标签的内容。
开始标签是英文，结束标签是中文。
